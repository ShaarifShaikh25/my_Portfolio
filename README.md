# My Portfolio

A personal portfolio website to showcase skills, projects, and contact details.

## Description

This is a responsive portfolio website built with HTML, CSS, and JavaScript. It features a modern design with smooth animations and a clean user interface to present my web development skills and projects.

## Features

- **Responsive Design**: Works seamlessly on all devices (desktop, tablet, mobile)
- **Smooth Animations**: Eye-catching transitions and hover effects
- **Contact Form**: Integrated contact form using Web3Forms
- **Project Showcase**: Display of my latest projects with links to live demos and GitHub
- **Social Links**: Easy access to GitHub, LinkedIn, and Instagram profiles

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (DOM manipulation, form handling)

## Sections

1. **Home**: Hero section with introduction and call-to-action buttons
2. **About**: Information about me and my skills
3. **Projects**: Showcase of my web development projects
4. **Contact**: Contact form for potential clients and collaborators

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ShaarifShaikh25/my_Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd my_Portfolio
   ```

3. Open `index.html` in your browser or use a local server.
